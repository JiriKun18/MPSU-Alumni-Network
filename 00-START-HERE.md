## 🚀 Quick Start Instructions

### For Windows Users:
```
1. Open Command Prompt
2. Navigate to project: cd d:\alumni system
3. Double-click install.bat
4. Answer prompts (press Y for database migration)
5. Run: php artisan serve
6. Visit: http://localhost:8000
```

### For Linux/Mac Users:
```bash
1. Open Terminal
2. Navigate to project: cd /path/to/alumni system
3. Run: bash install.sh
4. Answer prompts (type 'y' for database migration)
5. Run: php artisan serve
6. Visit: http://localhost:8000
```
